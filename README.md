# Usage

```
user1: Hi

user1: hubot note start // chat log save start
hubot: note "1997-01-12" start

user1: Hi Hi Hi!
user2: They can be demolished
user1: Hi Hi Hi!
user2: They can be toppled

user1: hubot note show // show saved last 3 chat log

hubot: user1: Hi Hi Hi!

```
